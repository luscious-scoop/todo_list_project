## To Do List

In this project I learned more about separating logic and UI and how to write code across multiple files. This project made me realize how much I still need to learn. The most difficult part of this project for me was local storage. I learned about how to store things in browser's local storage. Initially, my concept was entirely wrong: I was trying to store the toDo objects, which caused me a lot of problems and errors and took a lot of time(I was literally editing , deleting from the ls , instead of just clearing and adding the data to it), but I learned that I needed to store only the raw data itself and re add when changed, which solved a lot of my problems. I had to juggle around with local storage quite a bit. I also added finally learned how to add active tab functionality.I also with how add projects(a separate list ToDo's), but I pushed through and solved it too.
